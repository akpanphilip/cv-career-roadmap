## Quick Start
```bash
# Install dependencies
npm install

# Setup environment
cp .env.example .env.local
# Add your free Gemini API key from https://ai.google.dev/

# Check setup
npm run setup

# Run development server
npm run dev
```

Visit `http://localhost:3000`
